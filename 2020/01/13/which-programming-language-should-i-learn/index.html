Which Programming Language Should I Learn?

Making Your First Step Count

Published Mon, Jan 13, 2020 by Tony Towry

A student asked me what are the best programming languages for them to learn, if they are interested in #CyberSecurity? What say you, #Security peeps?

— Helen Patton (@OSUCISOHelen) January 11, 2020

Short answer python.

What if you want to overthink it a bit though.

The TIOBE Index bases it’s pulse off a number of inputs including YouTube, Baidu, Bing, and Google searches. Your mileage may vary, but this is still a useful clue for you regarding the rising and falling in popularity for any given language of interest.

Then there’s the Stack Overflow Developer Survey. Here you’ll see python doing quite well in both utility and joy.

So, learning python is a completely reasonable choice.

Why learn something else? Well, if you’re interested in appsec, learning a language with a widely used web framework makes good sense (yes, I’m aware of django and flask…but like, a real web framework). Try writing something with that framework, even if it’s just a throw away. Empathetic practice like this is invaluable.

Search Indeed for opportunities in information security that also contain the language you’re interested in. Do these seem like jobs you’d be interested in doing?

Finally, learn the language that you’ve got the most opportunity to use. There’s no substitute for spending time trying to solve problems with the tools you’ve got.

Now, go write some lines!
