Querying cvs Files with Headers

Default behavior is to ignore the first line and not use headers. You can change this by going to the storage tab in the web interface

Text Files: CSV, TSV, PSV - Apache Drill

Using COLUMNS[x] instead of columns[x]

Query Failed: An Error Occurred

org.apache.drill.common.exceptions.UserRemoteException: DATA_READ ERROR: Selected column ‘COLUMNS’ must have name ‘columns’ or must be plain ‘*’ File Path file:_sample-data_some.csv Fragment 0:0 [Error Id: 6e979344-63e0-4a9f-86ab-9bc0417637d3 on dev2:31010]

So, I got this one while working with a .csv after following what I thought the documentation had suggested with column indexing in the form of COLUMNS[x], but in actuality Drill wants columns[x].
