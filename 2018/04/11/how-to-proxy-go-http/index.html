How to Proxy Go net/http

Getting it over to ZAP

Published Wed, Apr 11, 2018 by Tony Towry

I started playing around a bit (again) in Go recently and had a need to take a look at some requests I was generating. I wanted to take a look in ZAP to see how things were working and use that to make adjustments, etc.

Post is no big deal, but I didn’t want to forget how to do this. Also, there are several posts that don’t seem to actually work.

First set the environment:

export HTTP_PROXY = localhost:8080

Then, use that as http.Transport. I’ve also skipped TLS verify for this as you can see.

 tr := &http.Transport{
  TLSClientConfig: &tlsConfig{InsecureSkipVerify: true},
  Proxy:  http.ProxyFromEnvironment,
 }

Finally, use that as the Transport for http.Client:

 client := &http.Client{Transport: tr}

Update: In my case I still preferred going through ZAP, but TIL if you just need to see the request it seems that you can also import net/http/httputil and just httputil.DumpRequest to take a look.
