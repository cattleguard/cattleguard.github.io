Playing With Buffalo - Part 0.2

Just some notes for things that I didn't see while I was googlin'

Published Sat, Aug 10, 2019 by Tony Towry

All notes are referencing buffalo 0.14.8 on Ubuntu.

One thing I’ve noticed is that the plugin framework may not be playing as described in the documentation. For example, installing the auth plugin.

buffalo plugins install github.com/gobuffalo/buffalo-auth

is supposed to, by my understanding, update the buffalo-plugins.toml file and go get the plugin for install. My experience however was that the command updated buffalo-plugins.toml, but did not install. Following tutorials and trying the g (generator) will result in an error: ERRO[0004] [PLUGIN] error loading plugin /home/username/go/bin/buffalo-auth: signal: killed

If the plugin was added to ./config/buffalo-plugins.toml then you can type buffalo plugins install and then buffalo plugins list. You should see your plugin properly installed.
